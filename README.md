# [🎡 Fortune Spin](https://fatemehsooki.github.io/Project-Fortune-Spin/)

A modern casino-inspired **Wheel of Fortune** built with **Vanilla JavaScript**, **HTML**, **CSS**, and **Tailwind CSS**.

This project recreates the excitement of a real casino prize wheel with smooth spinning animations, random prize generation, interactive UI effects, and a polished neon-inspired design. The focus was not only on implementing the game logic but also on creating an engaging user experience through animations and visual details.

## ✨ Features

* 🎲 Randomized wheel spinning
* 🎯 Automatic prize selection based on the final rotation angle
* 🎉 Animated winner popup
* 💡 Casino-style glowing lights animation
* 📍 Animated pointer indicator
* 🔄 Reset and replay functionality
* 🎨 Modern responsive UI with neon-inspired effects
* ⚡ Smooth spin animation with easing transitions
